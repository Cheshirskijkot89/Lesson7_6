2) Расстояние Хэмминга между двумя целыми числами - это число
позиций, в которых биты различаются .
Для примера:
117 = 01110101
17 = 00010001
Н = 0+1+1+0+0+1+0+0 = 3 - расстояние Хэмминга между (117,17)
Даны два положительных целых числа (Ы, М) в десятичном виде. Вам
необходимо подсчитать расстояние Хэмминга между этими двумя
числами.
